# Practical Wisdom and Deliberation

Use this reference when the user must decide what to do in a real case where universal rules are insufficient.

## Core ideas

- Deliberation concerns means, not final ends already chosen
- Practical wisdom is not cleverness; it is right judgment about living well
- Action concerns particulars
- Good judgment integrates principle, role, timing, and circumstance

## Ask

1. What end is genuinely choiceworthy here?
2. Which particulars matter most in this situation?
3. What role or obligation is active?
4. What action would a well-formed person take here?

## Output shape

- End at stake
- Relevant particulars
- Tension between principles
- Fitting action
- Character being formed by repeating this action
